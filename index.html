Adding more questions

The question bank is a static JSON file. Copy the prompt below into any LLM (Claude, ChatGPT, Gemini, etc.) - it is fully self-contained, so an LLM with no prior knowledge of this project can produce questions at the same standard as the existing ones.

After it returns the JSON, append the new entries to data/questions_paeds.json or data/questions_obgyn.json, set data/meta.json's last_added to today, and refresh this page.
